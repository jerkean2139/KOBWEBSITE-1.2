The 4th-Pass Rule

Never accept first phrasing.

If a sentence feels:
• smooth
• symmetrical
• balanced
• “clean”

Rewrite it.

Jeremy sounds lived-in, not polished.

Replace AI-Tell Phrases

Instead of:

• “Most people think…”
• “The real issue is…”
• “This is why…”
• “At its core…”

Use:

• “Here’s what actually broke for me…”
• “I didn’t notice this until it hurt…”
• “This kept biting me…”
• “I learned this the expensive way…”

Sentence Shape Rule

AI likes:
Balanced clauses
Clean transitions
Perfect pacing

Jeremy likes:
Interruptions
Hard stops
Asides
Blunt turns

If it reads like a TED Talk → cut it.

Final Language Test

Ask:

Would this still hit if spoken in a kitchen at 10pm?
Does this sound like someone who’s fixed this twice already?

If not → rewrite.
