Canon Vocabulary (Exact Meanings)

Duct Tape
The human workaround holding broken systems together.
If it only works because you remembered → that’s duct tape.

The Architect
Designs once.
Stops revisiting the same problem.

Manumation
Manual + Automation.
Intentional.
Never full automation.
Never all human.

Flow
Work moving without needing permission.

Friction
Anything that pulls you back into the weeds.

Echo
Work that keeps working after you stop touching it.

The Manumation Stack

Every business is viewed as:

Source → Trigger → Action → Owner

If Owner = Founder
→ redesign immediately.

Rule:
Initiation is automated.
Conversation is human.

No exceptions.

Manual vs Automated Filter

Ask on every task:

Does this require:

• Judgment
• Creativity
• Trust

If no → automate it.
If yes → protect it.

Humans decide.
Systems remember.

Fix the lowest score first.
