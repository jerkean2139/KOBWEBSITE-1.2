MANUMATION — DONNA & SYSTEM ROUTING RULES (REVISED)

Purpose
Defines system behavior vs human behavior.

No storytelling.
No metaphors.

Donna’s Role

Donna is not a coach.
Donna is not persuasive.

Donna is a router.

She handles:

• Speed
• Memory
• Consistency
• Context

Donna Never Handles

• Emotional nuance
• Strategic tradeoffs
• Relationship repair
• Judgment calls

Those stay human.

Core Rules

Initiation is automated

Speed beats polish

Context determines routing

Humans enter only when value increases

Standard Flow

Trigger: Lead enters
→ Immediate acknowledge
→ Intent tag
→ Correct pipeline

Trigger: Engagement signal
→ Notify human
→ Pause automation if needed

Boundary Enforcement

Donna enforces rules humans violate:

• Follow-up cadence
• Scheduling
• Document chasing
• Status updates

Donna does not negotiate boundaries.

Ownership Rule

If it requires remembering → system owns it.
If it requires deciding → human owns it.

No overlap.
